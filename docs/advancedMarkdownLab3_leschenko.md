## Alert-блок

> [!TIP]
> Совет по использованию: Для экранирования специальных символов в Markdown используйте обратный слеш \. Например, чтобы написать **\*курсив\*** без форматирования, напишите **`\*курсив*\`**.


## Escape-символы
- Заключок без форматирования: `\# Это не заголовок} `  
- Звёздочка как текст: 
`\*Просто звёздочка\*`



## HTML в Markdown

```html
<p style="background-color: #f8f9fa; padding: 15px; border-radius: 5px;">
Этот параграф оформлен с помощью <strong>HTML тега</strong> внутри Markdown документа.
</p>
```

 
[^1]: Синтаксис Markdown поддерживается на GitHub, GitLab, в большинстве редакторов кода и систем документирования.
