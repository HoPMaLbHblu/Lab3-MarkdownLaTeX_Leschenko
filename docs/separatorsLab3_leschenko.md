
Markdown — это легкий язык разметки, который позволяет форматировать текст с помощью простых символов, таких как решетки, звездочки и дефисы. Он был создан для того, чтобы писать структурированный контент было быстро и удобно, не отвлекаясь на сложные инструменты верстки. Исходный код на Markdown остается читаемым даже без преобразования в готовый документ.

---

Основные возможности Markdown
С помощью Markdown можно создавать заголовки разных уровней, выделять текст жирным или курсивным начертанием, формировать маркированные и нумерованные списки. Также поддерживаются вставка гиперссылок, изображений, цитат, таблиц и блоков с программным кодом. Благодаря простоте синтаксиса Markdown легко конвертируется в HTML, PDF и другие форматы, что делает его популярным инструментом для документации, заметок и веб-публикаций.

***

Пример форматирования:
Для создания заголовка используются решетки — одна решетка для самого крупного заголовка, две для подзаголовка и так далее.
# Заголовок 1
 
## Заголовок 2
  **Жирный текст оформляется двумя звездочками или двумя подчеркиваниями до и после фрагмента,** *а курсив — одной звездочкой*. 
  
  * Маркированные списки создаются с помощью звездочки или дефиса в начале строки, 
  
  1. нумерованные — с помощью цифры с точкой. 
  
  Cсылки записываются квадратными скобками для [текста] и круглыми для адреса, а изображения добавляются восклицательным знаком перед такой же конструкцией.
