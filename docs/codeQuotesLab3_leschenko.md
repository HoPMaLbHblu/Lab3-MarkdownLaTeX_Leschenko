### **Блоки кода**
Встроенный (inline код):
Команда `git status` показывает состояние репозитория.
Многострочный код:
```
print("Hello, Markdown!")
```
Можно указать язык: python, js, csharp и др.

### **Цитаты**

>Это цитата в Markdown.  
Вложенная цитата:  
Первый уровень
>> Второй уровень
