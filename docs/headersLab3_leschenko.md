
## Лаба номер 3. MarkdownLaTeX

#### Что такое Markdown Markdown — это легкий язык разметки, который позволяет форматировать текст с помощью простых символов, таких как решетки, звездочки и дефисы

#### Что такое LaTeX — это мощная система верстки и язык разметки, используемый преимущественно для создания научных, математических и технических документов высокого качества

#### Основные различия между Markdown и LaTeX Markdown ориентирован на простоту и быстроту написания текста для веб-публикаций, тогда как LaTeX предоставляет полный контроль над версткой и идеально подходит для сложных формул и академических работ

#### Области применения Markdown широко используется для написания документации, файлов README, постов в блогах и заметок, а LaTeX применяется в научных статьях, диссертациях, математических публикациях и книгах со сложной структурой
